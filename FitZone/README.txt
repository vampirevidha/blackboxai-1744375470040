# FitZone Web Application

## Overview
FitZone is a web application designed to provide users with information about gym classes, trainers, recipes, and more.

## Setup Instructions
1. Clone the repository.
2. Set up a web server (e.g., Apache, Nginx).
3. Configure the database connection in `db/config.php`.
4. Access the application via your web browser.

## Features
- User registration and login
- Class and trainer details
- Contact form for inquiries
- Recipe suggestions
- Blog posts and reviews
